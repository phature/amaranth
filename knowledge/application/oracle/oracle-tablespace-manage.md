# Oracle表空间管理

## 临时表空间创建

``` text
CREATE SMALLFILE TEMPORARY TABLESPACE A_TEMPORARY
  TEMPFILE '/opt/oracle/x86_64/database/server/12c/oradata/DEVELOP/A_TEMPORARY_001.db'
  SIZE 8M AUTOEXTEND ON NEXT 8M MAXSIZE 8G
  EXTENT MANAGEMENT LOCAL UNIFORM;
```

## 表空间创建

``` text
CREATE SMALLFILE TABLESPACE A_DEFAULT
  DATAFILE '/opt/oracle/x86_64/database/server/12c/oradata/DEVELOP/A_DEFAULT_001.db'
  SIZE 8M AUTOEXTEND ON NEXT 8M MAXSIZE 8G
  LOGGING
  DEFAULT NOCOMPRESS
  ONLINE
  EXTENT MANAGEMENT LOCAL AUTOALLOCATE
  SEGMENT SPACE MANAGEMENT AUTO;
```

## 表空间增加文件

``` text
ALTER TABLESPACE A_DEFAULT
  ADD DATAFILE '/opt/oracle/x86_64/database/server/12c/oradata/DEVELOP/A_DEFAULT_XXX.DB'
  SIZE 8M AUTOEXTEND ON NEXT 8M MAXSIZE 8G;
```
